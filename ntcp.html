Project 5: Network Topology Configuration Project

Our Network Topology Configuration Project involved creating different network topologies in Cisco Packet Tracer. We explored star, mesh, bus, and hybrid configurations and implemented concepts such as IP addressing, routing, and switch configuration. This project helped us understand the practical implementation of networking concepts.

Project Partners: Abdurrahmaan Shaikh & Rashad Khan.
