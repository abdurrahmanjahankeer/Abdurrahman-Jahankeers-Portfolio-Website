Project 1: Car Rental System

Our Car Rental System is a simple Tkinter user interface Python application that enables users to hire and return cars, and log all the rental history to a CSV file. We implemented it through object-oriented programming, with features of input validation, rental history, and support for regular and luxury cars.

Project Partners: Abdurrahmaan Shaikh, Rashad Khan & Dawood Khalid.
